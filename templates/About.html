Tecnología universal

T.U. tecnología Universal es una pequeña empresa dedicada al desarrollo de aplicaciones web, dispositivos IoT y aplicaciones moviles

SecurePassQR es un proyecto para controlar el acceso de entrada hacia la Universidad Tecnologíca de Tlaxcala y evitar el acceso de gente no perteneciente a la institución.


Todo esto gracias a 3 principales sistemas:

  • Aplicación móvil
  • Dispositivo IoT
  • Aplicación Web

    • Aplicación móvil

    La aplicación movil tiene la funcionalidad de que cada alumno cuente con una credencial virtual, con su códigoQR que le servira para dar acceso a las instalaciones de la universidad, siendo programada con el lenguaje de Dart y la paqueteria de Flutter



    Dispositivo IoT

    El dispositivo de IoT es un pequeño escaner en la entrada de la institución donde cada miembro del alumando podrá escanear su código QR al momento llegar para poder permitirle el acceso a la misma. Este útil dispositvo esta creado con un escaner "QR Code Scanning Module", una pantalla LCD, jumpers y una placa ESP32



    Aplicación Web

    La aplicación web es una herramienta que ayudara para descargar la app y las intrucciones para usarla



    Instrucciones para usar la aplicacion web


  • Descargar la app
    • Link de descargar

  • Iniciar sesión

    • Una vez abras la aplicacion puedes ingresa tu correo y contraseña en los campos correspondientes



  • Interfaz y menu

    • La pantalla inicial de la aplicacion mostrara tus datos basicos, en la parte izquierda encontraras un pequeño menu con otros apartados que te pueden servir



  • Codigo QR

    • En la pagina principal debajo de tus datos se encontrara tu codigo QR



  • Escanear Codigo QR

    • Deberas colocar la pantalla de tu celular enfrente del escaner para que pueda leerlo, en la pantalla del dispositivo se mostrara un mensaje de bienvenida
