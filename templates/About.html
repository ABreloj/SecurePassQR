Tecnología universal

T.U. tecnología Universal es una pequeña empresa dedicada al desarrollo de aplicaciones web, dispositivos IoT y aplicaciones moviles

SecurePassQR es un proyecto para controlar el acceso de entrada hacia la Universidad Tecnologíca de Tlaxcala y evitar el acceso de gente no perteneciente a la institución.


Todo esto gracias a 3 principales sistemas:

  • Aplicación móvil
  • Dispositivo IoT
  • Aplicación Web

    • Aplicación móvil

    La aplicación movil tiene la funcionalidad de que cada alumno cuente con una credencial virtual, con su códigoQR que le servira para dar acceso a las instalaciones de la universidad, siendo programada con el lenguaje de Dart y la paqueteria de Flutter



    Dispositivo IoT

    El dispositivo de IoT es un pequeño escaner en la entrada de la institución donde cada miembro del alumando, en este cada uno podrá escanear su código QR al momento llegar para poder permitirle el acceso a la misma. Este útil dispositvo esta creado con un escaner "QR Code Scanning Module", una pantalla LCD, jumpers y una placa ESP32



    Aplicación Web

    La aplicación web es una herramienta que ayudara a los administradores del proyecto, que les servira para poder ver los datos de los estudiantes, asi como agregar a nuevos o eliminar a aquellos que por cualquier motivo dejen de pertenecer a la institución
