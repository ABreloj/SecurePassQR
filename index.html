Tecnología universal

T.U. Tecnología Universal es una pequeña empresa dedicada al desarrollo de aplicaciones web, dispositivos IoT y aplicaciones móviles.

SecurePassQR es un proyecto para controlar el acceso de entrada hacia la Universidad Tecnológica de Tlaxcala y evitar el acceso de gente no perteneciente a la institución.


Todo esto gracias a 3 principales sistemas:

Aplicación móvil
Dispositivo IoT
Aplicación Web

Aplicación móvil

La aplicación móvil tiene la funcionalidad de que cada alumno cuente con una credencial virtual, con su código QR que le servirá para dar acceso a las instalaciones de la universidad, siendo programada con el lenguaje de Dart y la paquetería de Flutter.



Dispositivo IoT

El dispositivo de IoT es un pequeño escáner en la entrada de la institución donde cada miembro del alumnado podrá escanear su código QR al momento de llegar para poder permitirle el acceso a la misma. Este útil dispositivo está creado con un escáner "QR Code Scanning Module", una pantalla LCD, jumpers y una placa ESP32.



Aplicación Web

La aplicación web es una herramienta que ayudará para descargar la app y las instrucciones para usarla.



Instrucciones para usar la aplicación móvil

Link de descarga

Iniciar sesión

Una vez abras la aplicación, puedes ingresar tu correo y contraseña en los campos correspondientes.



Interfaz y menú

La pantalla inicial de la aplicación mostrará tus datos básicos; en la parte izquierda encontrarás un pequeño menú con otros apartados que te pueden servir.



Código QR

En la página principal debajo de tus datos se encontrará tu código QR.



Escanear Código QR

Deberás colocar la pantalla de tu celular frente al escáner para que pueda leerlo; en la pantalla del dispositivo se mostrará un mensaje de bienvenida.
